matrixbinary¶

class graphm.matrixbinary.MatrixBinary(**d)¶

Bases: graphm.amatrix.AMatrix

Manage a boolean matrix with binary lines

Increase performance of boolean matrices with usage of integers (binary) for representation of list of booleans for lines and columns

Caution

Instance variables

Variables

  • dimM (int) – number of rows

  • dimN (int) – number of columns

  • matrix (list) – matrix with integers representation of lines

  • matrixM (list/tuple) – contents binary integers for rows

  • matrixN (list/tuple) – contents binary integers for columns
